## 4. 등록/수정 팝업 (MDPC01P01)

팝업코드: MDPC01P01

### 팝업 필드

| 항목     | 컴포넌트 | 필수 | disabled 조건        | 비고               |
| -------- | -------- | ---- | -------------------- | ------------------ |
| 사업장   | Select   | ✅   | 수정 모드 시 disable |                    |
| 물류센터 | Select   | ✅   | 수정 모드 시 disable |                    |
| PC명     | Input    | ✅   |                      |                    |
| PC유형   | Select   | ✅   |                      |                    |
| IP주소   | Input    | ✅   |                      | IPv4 형식          |
| MAC주소  | Input    |      |                      |                    |
| 사용여부 | Select   |      |                      | 기본값 사용        |
| 비고     | Textarea |      |                      | 전체 너비(colspan) |

### 안내문구

※ IP주소는 동일 사업장·물류센터 내에서 중복 등록할 수 없습니다.
※ 사용여부가 [미사용]인 PC는 단말 로그인 선택 목록에서 제외됩니다.

---

## 5. 공통 업무규칙

| No. | 규칙 내용                                                            | 구분       |
| --- | ------------------------------------------------------------------- | ---------- |
| 1   | 동일 사업장+물류센터 내 동일 IP주소는 1건만 등록 가능하다           | 수량제한   |
| 2   | IP주소는 IPv4 형식(0~255 사이 4옥텟)만 입력 가능하다                | 값강제     |
| 3   | 수정 모드 시 사업장·물류센터는 변경할 수 없다                       | UI disable |
| 4   | 사용여부=미사용 PC는 단말 로그인 선택 목록에서 제외된다             | 특수기능   |
