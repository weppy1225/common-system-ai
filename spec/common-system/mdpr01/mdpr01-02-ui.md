## 4. 하단 좌측 — 대상품목 (D1)

### 4-1. 툴바 버튼

| 순번 | 버튼명 | 동작           | 비고 |
| ---- | ------ | -------------- | ---- |
| 1    | 행추가 | 빈 행 추가     |      |
| 2    | 행삭제 | 선택행 삭제    |      |
| 3    | 저장   | 변경사항 일괄저장 |   |

### 4-2. 그리드 컬럼

| 순번 | 컬럼명       | 정렬 | 비고                                                        |
| ---- | ------------ | ---- | ----------------------------------------------------------- |
| 1    | No.          | 중앙 |                                                             |
| 2    | checkbox     | 중앙 |                                                             |
| 3    | 대분류       | 중앙 | 인라인 편집(DropDown), 대상구분=대분류일 때만 편집 가능     |
| 4    | 대상품목번호 | 중앙 | 인라인 편집(Input + 팝업 🔍), 대상구분=품목일 때만 편집 가능 |
| 5    | 대상품목명   | 좌측 | 팝업/자동완성 선택 시 자동채움, 대상구분=품목일 때만 편집   |

### 4-3. 업무규칙

| 구분     | 내용                                              |
| -------- | ------------------------------------------------- |
| 수량제한 | 증정유형=여러번인 경우 D1 행은 1개만 등록 가능    |

## 5. 하단 우측 — 증정품목 (D2)

### 5-1. 툴바 버튼

| 순번 | 버튼명   | 동작                                   | 비고                                    |
| ---- | -------- | -------------------------------------- | --------------------------------------- |
| 1    | 행추가   | 빈 행 추가                             |                                         |
| 2    | 행삭제   | 선택행 삭제                            |                                         |
| 3    | 저장     | 변경사항 일괄저장                      |                                         |
| 4    | 금액복사 | 기준금액 배수 행 자동생성              | 파란색 버튼, 프로모션유형=품목일 때 비활성 |

### 5-2. 그리드 컬럼

| 순번 | 컬럼명       | 정렬 | 비고                                              |
| ---- | ------------ | ---- | ------------------------------------------------- |
| 1    | No.          | 중앙 |                                                   |
| 2    | checkbox     | 중앙 |                                                   |
| 3    | 적용금액/개수 | 우측 | 숫자 포맷, 직접입력, 금액복사 대상               |
| 4    | 증정품목번호 | 중앙 | 인라인 편집(Input + 팝업 🔍), 자동완성 지원       |
| 5    | 증정품목명   | 좌측 | 팝업/자동완성 선택 시 자동채움                    |
| 6    | 증정수량     | 우측 | 숫자 포맷, 직접입력                               |

### 5-3. 업무규칙

| 구분   | 내용                                                   |
| ------ | ------------------------------------------------------ |
| 값강제 | 프로모션유형=금액인 경우 증정유형은 한번만 선택 가능   |

## 6. 등록/수정 팝업 (MDPR01P01)

팝업코드: MDPR01P01

### 팝업 필드

| 항목         | 컴포넌트     | 필수 | disabled 조건                          | 비고                    |
| ------------ | ------------ | ---- | -------------------------------------- | ----------------------- |
| 사업장       | Select       | ✅   | 수정 모드 시 disable                   |                         |
| 프로모션명   | Input        | ✅   |                                        |                         |
| 프로모션유형 | Select       | ✅   |                                        |                         |
| 증정유형     | Select       | ✅   |                                        | 프로모션유형=금액이면 한번만 고정 |
| 대상구분     | Select       | ✅   |                                        |                         |
| 대상수량     | Input - 숫자 | ✅   |                                        |                         |
| 적용시작일자 | Date         | ✅   |                                        |                         |
| 적용종료일자 | Date         | ✅   |                                        |                         |
| 적용여부     | Select       |      |                                        |                         |
| 비고         | Textarea     |      |                                        | 전체 너비(colspan)      |

### 안내문구

※ 프로모션 유형이 [금액]인 경우 증정유형은 [한번]만 가능합니다.
※ 프로모션 등록 후 대상상품(D1)과 증정상품(D2)을 등록하세요.

## 7. 공통 업무규칙

| No. | 규칙 내용                                                                                       | 구분     |
| --- | ----------------------------------------------------------------------------------------------- | -------- |
| 1   | 프로모션을 먼저 등록한 후 대상품목(D1)·증정품목(D2)을 등록할 수 있다                          | 순서강제 |
| 2   | 증정유형=여러번인 경우 D1(대상품목)은 1개만 등록 가능하다                                      | 수량제한 |
| 3   | 프로모션유형=금액인 경우 증정유형은 한번만 선택 가능하다                                        | 값강제   |
| 4   | 대상구분=품목이면 [품목번호·품목명]만 활성화, 대상구분=대분류이면 [대분류]만 활성화            | UI disable |
| 5   | 금액복사: 기준금액×지정 개수만큼 배수 행 자동생성 (예: 30,000×10개 → 30,000/60,000…300,000) | 특수기능 |
