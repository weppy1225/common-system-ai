```
┌─────────────────────────────────────────────────────────────────────┐

│──────────────────────────│──────────────────────────────────────────│
│  사업장명*     [Input]    │  [ 물류센터 목록 그리드 ]                │
```

### 2-2. 사업장 선택

`Select` — 수정 가능 사업장 목록 자동 로드, 선택 시 좌측 폼 + 우측 그리드 갱신.
API: `GET /editable/bizs/{regBizSeq}` → [04-SQL `searchEditableBizs` §37](./mdbz01-04-be-mapper-sql.md)

### 2-4. 물류센터 그리드

#### 툴바 버튼

| 순번 | 버튼명 | 배치 | 동작 |
|---|---|---|---|
| 1 | 추가 | 좌측 | 그리드 빈 행 삽입, 입력 가능 스타일 적용 |
| 2 | 삭제 | 좌측 | 체크 행 삭제 마킹 |
| 3 | 저장 | 우측 헤더 | insert·update·delete 일괄 전송 |

#### 그리드 컬럼

| 순번 | 컬럼명 | 정렬 | 편집 | 비고 |
|---|---|---|---|---|
| 1 | No. | 중앙 | - | |
| 2 | checkbox | 중앙 | - | |
| 3 | 물류센터명 | 좌측 | O | 위탁 센터는 편집 불가 |
| 4 | 우편번호 | 중앙 | - | 🔍 클릭 → 카카오 주소 팝업, 위탁 센터는 🔍 미표시 |
| 5 | 주소 | 좌측 | - | 주소 검색으로만 변경 |
| 6 | 주소상세 | 좌측 | O | |
| 7 | 전화번호 | 중앙 | O | |
| 8 | 비고 | 좌측 | O | |
| 9 | 사용여부 | 중앙 | - | "사용"/"미사용" |

위탁 센터(거절 상태)는 행 흐리게 표시. 신규 추가 행은 입력 가능 스타일.

## 4. MDBZ01P02 — 물류대행업체검색 팝업 (mdbz01Sch.vue)

| 항목 | 내용 |
|---|---|
| 팝업코드 | MDBZ01P02 |
| 레이아웃 | 검색 조건 → 결과 그리드 → 의뢰자 정보 + 요청 내용 |

검색 파라미터 3개 (사업장명·물류센터명·주소, 모두 부분 검색) → [04-SQL `searchTplBizCenter` §26](./mdbz01-04-be-mapper-sql.md)

#### 검색 결과 그리드

| 순번 | 컬럼명 | 정렬 | 비고 |
|---|---|---|---|
| 1 | No. | 중앙 | |
| 2 | checkbox | 중앙 | 복수 선택 가능 |
| 3 | 요청상태 | 중앙 | 신청가능·신청중·승인·거절 (상태 정의 → [03-data-model §3-1](./mdbz01-03-data-model.md)) |
| 4 | 사업장명 | 좌측 | 동일 사업장 복수 센터 병합 표시 |
| 5 | 센터명 | 좌측 | |
| 6 | 주소 | 좌측 | 주소 + 주소상세 합산 표시 |
| 7 | 전화번호 | 중앙 | |
| 8 | 이메일 | 중앙 | |

컬럼 필터 지원.

의뢰자 정보(읽기 전용): 로그인 사용자의 사업장명·담당자명·전화번호 자동 표시 + 요청내용 Textarea 입력.

#### 버튼

| 버튼 | 조건 | 동작 |
|---|---|---|
| 검색 | 항상 | 검색 조건으로 대행 업체 검색 |
| 초기화 | 항상 | 검색 조건 초기화 |
| 위탁 요청 | 1건 이상 체크 | 체크 센터에 위탁 의뢰 신청. 신청중·승인 상태 중복 차단. 완료 후 팝업 닫기 + 부모 갱신 |
