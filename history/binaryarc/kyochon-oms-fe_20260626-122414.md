# 작업 히스토리

## SHPD01 팝업 버튼 제거 + 그리드 HOT/NEW/전시여부 공통코드 적용

### 변경 내용

1. **`shpd01Edt.vue` — 목록·삭제 버튼 제거**
   - 이전: button-wrapper에 목록(gray), 삭제(red, v-if=isUpdate), 등록/수정(blue) 3개 버튼
   - 이후: 등록/수정(blue) 버튼만 남김
   - 사유: 팝업에서 목록·삭제 기능 제거 요건
   - 연동 제거: `vfn_delete` 함수 삭제 (팝업 삭제 버튼 삭제로 사용처 없음)

2. **`shpd01.vue` — 그리드 공통코드 변환 추가 및 dataField 변경**
   - 이전: `USE_YN`으로 `dispYn` 변환, `hotYn`·`newYn` 코드값 그대로 표시
   - 이후:
     - `SHOP_DISP_YN` → `dispYn` → `dispYnNm` (전시여부)
     - `HOT_YN` → `hotYn` → `hotYnNm` (인기상품 여부)
     - `NEW_YN` → `newYn` → `newYnNm` (신규상품 여부)
   - 그리드 dataField: `hotYn` → `hotYnNm`, `newYn` → `newYnNm`, `dispYn` → `dispYnNm`
   - 사유: Y/N 코드값 대신 명칭(인기/보통, 신규/기존, 전시/비전시) 표시

### DB 공통코드 등록 (dev)
- `sm_comm_h` + `sm_comm_d`: `HOT_YN` (인기상품_여부) — Y:인기, N:보통
- `sm_comm_h` + `sm_comm_d`: `NEW_YN` (신규상품_여부) — Y:신규, N:기존
- `SHOP_DISP_YN`은 기존 등록 완료 상태
