# 작업 히스토리

## 파일 지도 생성기에서 history/ 폴더 제외

### 변경 내용
1. **`scripts/gen-md-map.py` os.walk 가지치기에 `history` 추가**
   - 이전: `dns[:] = [d for d in dns if d not in ('node_modules', '.git')]`
   - 이후: `dns[:] = [d for d in dns if d not in ('node_modules', '.git', 'history')]`
   - 사유: 작업 히스토리 로그(history/)가 파일 지도(20-md-index.html) 인덱스에 같이 집계되어, 문서 지도에 작업 로그가 섞여 노이즈가 됨. history는 추적용 로그라 문서 인덱스 대상이 아님.

2. **`knowledgebase/20-md-index.html` 재생성**
   - 이전: md 221개 (history 26개 포함)
   - 이후: md 195개 (history 제외)
   - 검증: `grep -c "history/"` → 0
