# 작업 히스토리

## 파일·폴더 지도 HTML 생성 위치 변경 (knowledgebase → edu, 교육자료화)

### 변경 내용
1. **생성기 출력 경로 변경 `scripts/gen-md-map.py`**
   - 이전: `knowledgebase/20-md-index.html` 에 출력
   - 이후: `edu/md-index.html` 에 출력 (`os.makedirs('edu', exist_ok=True)` 추가)
   - 사유: 파일 지도를 교육자료(`edu/`)로 관리하기 위함. 헤더 주석의 부정확한 "gitignore 대상" 표기도 실제(추적 파일)에 맞게 "교육자료, 생성물" 로 수정.

2. **추적 파일 이동 (git mv)**
   - 이전: `knowledgebase/20-md-index.html`
   - 이후: `edu/md-index.html`
   - 사유: 기존 생성물이 git 추적 상태였으므로 이력 보존하며 이동.

3. **문서 포인터 일괄 수정 (5개 파일)**
   - `STRUCTURE.md` — scripts 설명·knowledgebase 트리에서 html 항목 제거 후 새 위치 안내
   - `README.md` — 파일 지도 링크 `edu/md-index.html`
   - `knowledgebase/20-md-index.md` — description·본문 포인터
   - `.claude/skills/md_index/SKILL.md` — description·when_to_use·본문·검증 명령 경로
   - 사유: `knowledgebase/20-md-index.html` 을 가리키던 라이브 참조를 새 경로로 정합화 (history/ 과거 기록은 수정하지 않음).

### 검증
- `python scripts/gen-md-map.py` 재실행 → `edu/md-index.html` 생성 성공 (md 196개), 옛 경로 미생성 확인.
- `python scripts/check-doc-refs.py` → 깨진 참조 0, 미참조 패턴 0, BOM 0 (종료코드 0).
