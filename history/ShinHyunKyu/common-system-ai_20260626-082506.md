# 작업 히스토리

## 다른 AI 코딩 도구(Codex·Cursor·Copilot·Gemini) 지원 암시 내용 일괄 제거

지원 대상은 Claude Code 단일로 하고, 다른 AI 도구를 지원하는 것처럼 보이는 문구를 전 md 파일에서 제거했다. (CSS `cursor:` 속성은 도구 무관이라 제외)

### 변경 내용
1. **CLAUDE.md (3행)**
   - 이전: `AI 에이전트(Claude Code·Codex)가 ...`
   - 이후: `AI 에이전트(Claude Code)가 ...`
   - 사유: Codex 지원 암시 제거

2. **.claude/rules/md-frontmatter.md**
   - 35행: `Claude Code와 Codex가 공식적으로 해석하는...` → `Claude Code가 공식적으로 해석하는...`
   - 46행: `Claude Code나 Codex의 공식 frontmatter...` → `Claude Code의 공식 frontmatter...`
   - 101~103행: Codex Skill/App UI/AGENTS.md 관련 불릿 3개 줄 전체 삭제
   - 110행: `Claude Code/Codex 공식 필드...` → `Claude Code 공식 필드...`
   - 사유: Codex 공식 스키마 설명 제거

3. **.claude/rules/rule-skill-frontmatter.md**
   - 33행: `Cursor 의 alwaysApply: true 와 동일한 효과.` 문구 삭제
   - 183행: `alwaysApply 같은 Cursor 전용 필드를...` → `alwaysApply 같은 비공식 필드를...`
   - 사유: Cursor 도구 언급 제거 (규칙 의도는 유지)

4. **.claude/skills/PI_411/SKILL.md**
   - 90행: `.claude/, .agents/, .codex/, .cursor/` → `.claude/`
   - 91행: `CLAUDE.md, AGENTS.md, COPILOT.md, GEMINI.md` → `CLAUDE.md`
   - 442행: 제외 정책에서 `.agents/`, `AGENTS.md` 제거
   - 사유: ZIP 패키징 제외 목록에서 타 AI 도구 산출물 항목 제거 (사용자 A안 선택)
