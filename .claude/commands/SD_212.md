# 테이블정의서 생성 명령어

업체명: **$ARGUMENTS**

BE 프로젝트의 테이블 정의 MD 파일들을 읽어  
`template/03 설계(SD)/SD.212-테이블정의서.xlsx` 양식으로 산출물을 생성한다.

---

## 입출력 경로

| 구분 | 경로 |
|---|---|
| MD 소스 | `C:\zinide\workspace_cloud\cloud-wms-be\DEV_DOC\ai-docs\10-database\90-schema\20-tables\` |
| 템플릿 | `template\03 설계(SD)\SD.212-테이블정의서.xlsx` |
| 출력 | `output\03 설계(SD)\SD.212-테이블정의서_{업체명}_{YYMMDD}.xlsx` |

- `{업체명}`: $ARGUMENTS
- `{YYMMDD}`: 오늘 날짜 (예: 260506)
- 출력 폴더가 없으면 생성

---

## MD 파일 구조 (파싱 기준)

```
# {physical_table_name} ({logical_table_name})   ← 1행에서 테이블명 추출

## 1. 개요
{개요 텍스트}                                    ← Table List Remark, 시트 Remark에 사용

## 2. 테이블 정의
| PK/FK | 컬럼명 | 타입 | NULL | 기본값 | 설명 |
|---|---|---|---|---|---|
| PK | col_name | varchar(50) | N | | 설명 |   ← 컬럼 데이터

## 3. 인덱스
| 인덱스명 | 컬럼 | UNIQUE | PK |

## 5. FK 관계
| FK 컬럼 | 참조 테이블 | 참조 컬럼 | 제약명 |

## 6. 참조됨 (참조하는 테이블)
| 참조 테이블 | 참조 컬럼 | 제약명 |
```

## 실행 절차

### 1단계 — MD 파일 전체 파싱

PowerShell로 MD 파일을 파싱하는 함수를 작성한다.

```powershell
function Parse-TableMD {
    param([string]$FilePath)

    $result = @{
        PhysicalName = ""; LogicalName = ""; Overview = ""
        Columns = @(); Indexes = @(); FKs = @(); RefBy = @()
    }

    $content = Get-Content $FilePath -Encoding UTF8
    
    # 첫 비공백 행에서 테이블명 추출: # physical_name (논리명)
    foreach ($line in $content) {
        if ($line -match "^#\s+(\S+)\s+\((.+)\)") {
            $result.PhysicalName = $Matches[1]; $result.LogicalName = $Matches[2]; break
        } elseif ($line -match "^#\s+(\S+)") {
            $result.PhysicalName = $Matches[1]; $result.LogicalName = $Matches[1]; break
        }
    }

    $section = ""; $overviewLines = @()
    foreach ($line in $content) {
        if ($line -match "^##\s+1\.") { $section = "overview"; continue }
        if ($line -match "^##\s+2\.") { $section = "columns";  continue }
        if ($line -match "^##\s+3\.") { $section = "indexes";  continue }
        if ($line -match "^##\s+[456789]") {
            if ($line -match "FK 関係|FK 관계") { $section = "fks" }
            elseif ($line -match "참조됨") { $section = "refby" }
            else { $section = "other" }
            continue
        }
        if ($line -match "^#[^#]") { continue }
        if ($line -match "^>|^```") { continue }

        if ($line -match "^\|") {
            if ($line -match "^\|[-\s|]+\|$") { continue }  # 구분선
            $cells = $line -split '\|' | ForEach-Object { $_.Trim() } | Where-Object { $_ -ne "" }

            if ($section -eq "columns" -and $cells.Count -ge 6) {
                if ($cells[0] -eq "PK/FK") { continue }
                $result.Columns += @{
                    PKFK     = $cells[0]
                    Physical = $cells[1]
                    DataType = $cells[2]
                    Nullable = $cells[3]
                    Default  = if ($cells.Count -gt 4) { $cells[4] } else { "" }
                    Logical  = if ($cells.Count -gt 5) { $cells[5] } else { "" }
                }
            } elseif ($section -eq "indexes" -and $cells.Count -ge 2) {
                if ($cells[0] -eq "인덱스명" -or $cells[0] -eq "인덱스 명") { continue }
                $result.Indexes += @{
                    Name    = $cells[0]
                    Columns = if ($cells.Count -gt 1) { $cells[1] } else { "" }
                    Unique  = if ($cells.Count -gt 2) { $cells[2] } else { "" }
                    PK      = if ($cells.Count -gt 3) { $cells[3] } else { "" }
                }
            } elseif ($section -eq "fks" -and $cells.Count -ge 2) {
                if ($cells[0] -eq "FK 컬럼") { continue }
                $result.FKs += @{
                    Column   = $cells[0]
                    RefTable = if ($cells.Count -gt 1) { $cells[1] } else { "" }
                    RefCol   = if ($cells.Count -gt 2) { $cells[2] } else { "" }
                    Const    = if ($cells.Count -gt 3) { $cells[3] } else { "" }
                }
            } elseif ($section -eq "refby" -and $cells.Count -ge 2) {
                if ($cells[0] -eq "참조 테이블") { continue }
                $result.RefBy += @{
                    Table  = $cells[0]
                    Column = if ($cells.Count -gt 1) { $cells[1] } else { "" }
                }
            }
            continue
        }
        if ($section -eq "overview" -and $line.Trim() -ne "") {
            $overviewLines += $line.Trim()
        }
    }

    $ov = ($overviewLines -join " ")
    $result.Overview = if ($ov.Length -gt 200) { $ov.Substring(0, 200) } else { $ov }
    return $result
}
```

MD 파일을 그룹 순서로 정렬하여 파싱한다.

```powershell
$tableDir = "C:\zinide\workspace_cloud\cloud-wms-be\DEV_DOC\ai-docs\10-database\90-schema\20-tables"
$mdFiles  = Get-ChildItem $tableDir -Filter "*.md" | Sort-Object {
    switch -Wildcard ($_.BaseName) {
        "mdm_*" { "1_$($_.BaseName)" }
        "sif_*" { "2_$($_.BaseName)" }
        "sm_*"  { "3_$($_.BaseName)" }
        "wes_*" { "4_$($_.BaseName)" }
        "wms_*" { "5_$($_.BaseName)" }
        default  { "9_$($_.BaseName)" }

### 2단계 — 출력 파일 준비

```powershell
$projectRoot  = "C:\zinide\workspace_cloud\cloud-wms-doc"
$templatePath = "$projectRoot\template\03 설계(SD)\SD.212-테이블정의서.xlsx"
$outputDir    = "$projectRoot\output\03 설계(SD)"
$yymmdd       = (Get-Date).ToString("yyMMdd")
$outputPath   = "$outputDir\SD.212-테이블정의서_$ARGUMENTS_$yymmdd.xlsx"

if (-not (Test-Path $outputDir)) { New-Item -ItemType Directory -Path $outputDir -Force }
Copy-Item -Path $templatePath -Destination $outputPath -Force

$excel = New-Object -ComObject Excel.Application
$excel.Visible = $false
$excel.DisplayAlerts = $false
$wb = $excel.Workbooks.Open($outputPath)
```

### 3단계 — Table List 시트 작성

- 기존 데이터(R3~) 삭제
- 파싱된 테이블 목록을 순서대로 입력 (No / 논리명 / 물리명 / 개요)
- 헤더(R2) 배경색 연두색 유지

### 4단계 — 기존 테이블 시트 삭제 후 재생성

- Table List를 제외한 기존 시트 전체 삭제
- 각 테이블마다 시트를 새로 추가하고 위 구조대로 데이터 입력
- 시트명: 물리 테이블명 (31자 초과 시 잘라냄, 중복 시 뒤에 숫자 추가)
- 배경색 연두색 (`Interior.Color = 11396999`): 섹션 헤더 레이블 열(A열), 컬럼 헤더(R13)

### 5단계 — 인쇄 최적화 (모든 시트)

아래 설정을 **모든 시트**에 적용한다.

#### 테두리 (xlContinuous=1, xlThin=2, xlMedium=-4138)

```
경계 번호 참조:
  7=xlEdgeLeft, 8=xlEdgeTop, 9=xlEdgeBottom, 10=xlEdgeRight
  11=xlInsideVertical, 12=xlInsideHorizontal
```

- 데이터 있는 전체 범위: 얇은 실선 (`xlThin`)
- 섹션 시작 행 상단 (`Table info` / `Column info` / `Index info` / `FK info` / `FK info (PK Side)`): 굵은선 (`xlMedium`)
