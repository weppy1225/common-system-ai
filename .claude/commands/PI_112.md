# 프로그램 목록 생성 명령어

업체명: **$ARGUMENTS**

BE / FE 소스 ZIP을 스캔하고 sm_menu DB를 조회하여 PI.112-프로그램목록 Excel을 생성한다.

---

## 입출력 경로

| 구분 | 경로 |
|---|---|
| 설정 파일 | `.claude/config/github.json` (DB 접속 정보: `application-test.properties` 참조) |
| BE ZIP | `output\04 구현(PI)\PI.111 프로그램 소스\cloud-wms-be-main_{업체명}_*.zip` |
| FE ZIP | `output\04 구현(PI)\PI.111 프로그램 소스\cloud-wms-fe-main_{업체명}_*.zip` |
| 템플릿 | `template\04 구현(PI)\PI.112-프로그램목록.xlsx` |
| 출력 | `output\04 구현(PI)\PI.112-프로그램목록_{업체명}_{YYMMDD}.xlsx` |

- `{업체명}`: $ARGUMENTS
- `{YYMMDD}`: 오늘 날짜

---

## 소스 구조

### BE (`프로그램목록_BE` 시트)
```
src/main/java/be/{lv2}/{lv3}/[bean/]{ClassName}.java|xml
  lv2 = 그룹코드 (예: iv3000, comm)
  lv3 = 메뉴코드 (예: ivad01) — comm 그룹은 lv3 없음
  C8  = lv3 (메뉴코드 소문자) 또는 comm
  C9  = sm_menu.menu_nm 조회
  C10 = 파일명 (확장자 제거)
  C11 = 모듈설명 (클래스명 패턴으로 자동 생성)
  C12 = java | xml
```

### FE (`프로그램목록_FE` 시트)
```
src/views/be/{lv4}/{lv5}/{fileName}.vue
src/components/be/{lv4}/{lv5}/{fileName}.vue|js
src/api/{fileName}.js|ts
src/assets/...
src/router/...
src/stores/...
src/layout/...
  C7  = lv5 (메뉴코드 소문자) 또는 파일명
  C8  = sm_menu.menu_nm 또는 공통
  C9  = 파일명 (확장자 제거)
  C10 = 모듈설명 (파일명 패턴으로 자동 생성)
  C11 = vue | js | ts | scss | css
```

---

## 주의사항

- 모든 단계(스캔 + Excel 생성)는 **단일 PowerShell 블록**으로 실행한다. 세션이 분리되면 스캔 데이터가 초기화되어 0건이 입력된다.
- BE ZIP / FE ZIP이 없으면 `/PI_111 {업체명}` 먼저 실행한다.

### 4단계 — 모듈설명 생성 함수 정의

```powershell
function Get-BeModuleDesc {
    param([string]$menuNm, [string]$className, [string]$subFolder)
    $nm = $menuNm

    # bean 하위 객체
    if ($subFolder -eq 'bean') {
        if ($className -match 'Request$')       { return "$nm 요청 객체" }
        if ($className -match 'Response$')      { return "$nm 응답 객체" }
        if ($className -match 'SearchDetail$')  { return "$nm 상세검색 객체" }
        if ($className -match 'Search$')        { return "$nm 검색 객체" }
        if ($className -match 'Excel$')         { return "$nm 엑셀 객체" }
        if ($className -match 'Label$')         { return "$nm 라벨 객체" }
        if ($className -match 'Paper$')         { return "$nm 지시서 객체" }
        if ($className -match 'Prod(Info)?$')   { return "$nm 품목 객체" }
        if ($className -match 'Info$')          { return "$nm 정보 객체" }
        if ($className -match '(Tran|Process|Save|Cancel|With)') { return "$nm 처리 객체" }
        return "$nm 객체"
    }

    # 일반 클래스
    if ($className -match 'CompUtil$')    { return "$nm CompUtil" }
    if ($className -match 'TxComp$')      { return "$nm Comp" }
    if ($className -match 'Comp$')        { return "$nm Comp" }
    if ($className -match 'Controller$')  { return "$nm Controller" }
    if ($className -match 'Dao$')         { return "$nm Dao" }
    if ($className -match 'Mapper$')      { return "$nm Mapper" }
    return "$nm $className"
}

function Get-FeModuleDesc {
    param([string]$menuNm, [string]$fileName)
    $nm = if ($menuNm) { $menuNm } else { '공통' }
    $fn = $fileName.ToLower()

    if ($fn -match 'edt$')        { return "$nm 수정" }
    if ($fn -match 'add$')        { return "$nm 등록" }
    if ($fn -match 'print')       { return "$nm 출력" }
    if ($fn -match 'popup')       { return "$nm 팝업" }
    if ($fn -match 'detail')      { return "$nm 상세" }
    if ($fn -match 'list$')       { return "$nm 목록" }
    return "$nm"
}
```

### 5단계 — BE 파일 스캔

```powershell
$beJavaRoot = "$beExtract\cloud-wms-be-main\src\main\java"
$beRows = [System.Collections.Generic.List[object]]::new()

$beFiles = Get-ChildItem $beJavaRoot -Recurse -File |
    Where-Object { $_.Extension -in '.java', '.xml' } |
    Sort-Object FullName

foreach ($file in $beFiles) {
    $rel   = $file.FullName.Replace($beJavaRoot + '\', '')  # be\iv3000\ivad01\IVAD01Comp.java
    $parts = $rel.Split('\')
    if ($parts.Count -lt 2) { continue }

    $lv1 = $parts[0]   # be
    $lv2 = $parts[1]   # iv3000 | comm | bm | ...
    $ext  = $file.Extension.TrimStart('.')

    $lv3=$null; $lv4=$null; $progId=$null; $progNm=$null; $subFolder=$null

    if ($lv2 -eq 'comm') {
        # 공통: be\comm\[bean\]ClassName
        $progId = 'comm'; $progNm = '공통'
        if ($parts.Count -ge 4) { $lv3=$parts[2]; $subFolder=$parts[2] }
        elseif ($parts.Count -ge 3 -and $parts[2] -notmatch '\.' ) { $lv3=$parts[2]; $subFolder=$parts[2] }
    } else {
        # 일반 메뉴: be\{group}\{menu}\[bean\]ClassName
        if ($parts.Count -ge 3) {
            $lv3 = $parts[2]
            if ($parts.Count -ge 4 -and $parts[3] -notmatch '\.') {
                $lv4 = $parts[3]; $subFolder = $parts[3]
            }
            # sm_menu lookup
            $urlKey = "/$lv1/$lv2/$lv3"
            $menuRow = $menuData | Where-Object { $_.menu_url -eq $urlKey } | Select-Object -First 1
            if (-not $menuRow) {
                # 대소문자 무시 재시도
                $menuRow = $menuData | Where-Object { $_.menu_url -ieq $urlKey } | Select-Object -First 1
            }
            $progId = $lv3
            $progNm = if ($menuRow) { $menuRow.menu_nm } else { $lv3 }
        }
    }

    $className = [System.IO.Path]::GetFileNameWithoutExtension($file.Name)
    $menuNmForDesc = if ($lv2 -eq 'comm') { '공통기능' } else { $progNm }
    $desc = Get-BeModuleDesc -menuNm $menuNmForDesc -className $className -subFolder $subFolder

    $beRows.Add([pscustomobject]@{
        lv1=$lv1; lv2=$lv2; lv3=$lv3; lv4=$lv4; lv5=$null; lv6=$null; lv7=$null
        progId=$progId; progNm=$progNm; modulNm=$className; desc=$desc; devType=$ext
        reqId=$null; remark=$null
    })
}

Write-Host "BE 파일 수: $($beRows.Count)"
```

### 6단계 — FE 파일 스캔

```powershell
$feSrcRoot = "$feExtract\cloud-wms-fe-main\src"
$feRows = [System.Collections.Generic.List[object]]::new()

$extFilter = @('.vue','.js','.ts','.scss','.css')
$feFiles = Get-ChildItem $feSrcRoot -Recurse -File |
    Where-Object { $_.Extension -in $extFilter } |
    Sort-Object FullName

foreach ($file in $feFiles) {
    $rel   = $file.FullName.Replace($feSrcRoot + '\', '')  # views\be\iv3000\ivad01\ivad01.vue
    $parts = $rel.Split('\')
    $ext   = $file.Extension.TrimStart('.')
    $fileName = [System.IO.Path]::GetFileNameWithoutExtension($file.Name)

    # Lv 분배 (최대 6단계)
    $lvs = @($null,$null,$null,$null,$null,$null)
    $maxLv = [math]::Min($parts.Count - 1, 5)  # 마지막은 파일명이므로 제외
    for ($i = 0; $i -lt $maxLv; $i++) { $lvs[$i] = $parts[$i] }

    $lv1=$lvs[0]; $lv2=$lvs[1]; $lv3=$lvs[2]; $lv4=$lvs[3]; $lv5=$lvs[4]; $lv6=$lvs[5]

    # 메뉴명 결정
    $progId=$null; $progNm=$null

    if (($lv2 -eq 'views' -or $lv2 -eq 'components') -and $lv3 -eq 'be' -and $lv5) {
        # views/be/{group}/{menu}/
        $urlKey = "/be/$lv4/$lv5"
        $menuRow = $menuData | Where-Object { $_.menu_url -ieq $urlKey } | Select-Object -First 1
        $progId = $lv5
        $progNm = if ($menuRow) { $menuRow.menu_nm } else { $lv5 }
    } elseif (($lv2 -eq 'views' -or $lv2 -eq 'components') -and $lv3 -eq 'bm' -and $lv5) {
        $urlKey = "/bm/$lv4/$lv5"
        $menuRow = $menuData | Where-Object { $_.menu_url -ieq $urlKey } | Select-Object -First 1
        $progId = $lv5
        $progNm = if ($menuRow) { $menuRow.menu_nm } else { $lv5 }
    } elseif ($lv2 -eq 'components' -and $lv3 -eq 'be' -and -not $lv5) {
        $progId = $fileName; $progNm = '공통'
    } else {
        $progId = $fileName; $progNm = '공통'
    }

    $desc = Get-FeModuleDesc -menuNm $progNm -fileName $fileName

    $feRows.Add([pscustomobject]@{
        lv1=$lv1; lv2=$lv2; lv3=$lv3; lv4=$lv4; lv5=$lv5; lv6=$lv6
        progId=$progId; progNm=$progNm; modulNm=$fileName; desc=$desc; devType=$ext
        reqId=$null; remark=$null
    })
}

Write-Host "FE 파일 수: $($feRows.Count)"
```

### 7단계 — Excel 생성

```powershell
Copy-Item -Path $templatePath -Destination $outputPath -Force

$excel = New-Object -ComObject Excel.Application
$excel.Visible = $false; $excel.DisplayAlerts = $false
$wb = $excel.Workbooks.Open($outputPath)

# --- BE 시트 ---
$wsBe = $wb.Worksheets.Item("프로그램목록_BE")
$lastRow = $wsBe.Cells($wsBe.Rows.Count, 1).End(-4162).Row
if ($lastRow -ge 3) {
    $wsBe.Range($wsBe.Rows.Item(3), $wsBe.Rows.Item($lastRow)).Delete() | Out-Null
}

$r = 3
foreach ($row in $beRows) {
    $wsBe.Cells.Item($r, 1).Value2  = $row.lv1
    $wsBe.Cells.Item($r, 2).Value2  = $row.lv2
    $wsBe.Cells.Item($r, 3).Value2  = $row.lv3
    $wsBe.Cells.Item($r, 4).Value2  = $row.lv4
    $wsBe.Cells.Item($r, 5).Value2  = $row.lv5
    $wsBe.Cells.Item($r, 6).Value2  = $row.lv6
    $wsBe.Cells.Item($r, 7).Value2  = $row.lv7
    $wsBe.Cells.Item($r, 8).Value2  = $row.progId
    $wsBe.Cells.Item($r, 9).Value2  = $row.progNm
    $wsBe.Cells.Item($r, 10).Value2 = $row.modulNm
    $wsBe.Cells.Item($r, 11).Value2 = $row.desc
    $wsBe.Cells.Item($r, 12).Value2 = $row.devType
    $r++
}

# --- FE 시트 ---
$wsFe = $wb.Worksheets.Item("프로그램목록_FE")
$lastRowFe = $wsFe.Cells($wsFe.Rows.Count, 1).End(-4162).Row
if ($lastRowFe -ge 3) {
    $wsFe.Range($wsFe.Rows.Item(3), $wsFe.Rows.Item($lastRowFe)).Delete() | Out-Null
}

$r = 3
foreach ($row in $feRows) {
    $wsFe.Cells.Item($r, 1).Value2  = $row.lv1
    $wsFe.Cells.Item($r, 2).Value2  = $row.lv2
    $wsFe.Cells.Item($r, 3).Value2  = $row.lv3
    $wsFe.Cells.Item($r, 4).Value2  = $row.lv4
    $wsFe.Cells.Item($r, 5).Value2  = $row.lv5
    $wsFe.Cells.Item($r, 6).Value2  = $row.lv6
    $wsFe.Cells.Item($r, 7).Value2  = $row.progId
    $wsFe.Cells.Item($r, 8).Value2  = $row.progNm
    $wsFe.Cells.Item($r, 9).Value2  = $row.modulNm
    $wsFe.Cells.Item($r, 10).Value2 = $row.desc
    $wsFe.Cells.Item($r, 11).Value2 = $row.devType
    $r++
}

$wb.Save()
$wb.Close($false)
$excel.Quit()
[System.Runtime.Interopservices.Marshal]::ReleaseComObject($excel) | Out-Null
[GC]::Collect()

$fileSize = [math]::Round((Get-Item $outputPath).Length / 1KB, 1)
Write-Host "저장 완료: $outputPath ($fileSize KB)"
```
