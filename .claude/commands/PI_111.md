# 프로그램 소스 ZIP 생성 명령어

업체명: **$ARGUMENTS**

GitHub에서 BE / FE 소스를 ZIP으로 다운로드하여 산출물 폴더에 저장한다.

---

## 입출력 경로

| 구분 | 경로 |
|---|---|
| 설정 파일 | `.claude/config/github.json` |
| 출력 | `output\04 구현(PI)\PI.111 프로그램 소스\cloud-wms-be-main_{업체명}_{YYMMDD}.zip` |
| 출력 | `output\04 구현(PI)\PI.111 프로그램 소스\cloud-wms-fe-main_{업체명}_{YYMMDD}.zip` |

- `{업체명}`: $ARGUMENTS
- `{YYMMDD}`: 오늘 날짜 (예: 260506)
- 출력 폴더가 없으면 자동 생성

---

## 실행 절차

### 1단계 — 설정 읽기

```powershell
$projectRoot = "C:\zinide\workspace_cloud\cloud-wms-doc"
$configPath  = "$projectRoot\.claude\config\github.json"
$config      = Get-Content $configPath -Encoding UTF8 | ConvertFrom-Json

$token    = $config.token
$beUrl    = $config.repos.be
$feUrl    = $config.repos.fe
$branch   = $config.branch
$company  = "$ARGUMENTS"
$yymmdd   = (Get-Date).ToString("yyMMdd")

$outputDir = "$projectRoot\output\04 구현(PI)\PI.111 프로그램 소스"
if (-not (Test-Path $outputDir)) { New-Item -ItemType Directory -Path $outputDir -Force }
```

### 2단계 — BE 소스 다운로드

```powershell
# owner/repo 추출 (예: ZinCoAtoZ/cloud-wms-be)
$beRepo    = ($beUrl -replace '\.git$','') -replace 'https://github.com/',''
$beZipUrl  = "https://codeload.github.com/$beRepo/zip/refs/heads/$branch"
$beOutPath = "$outputDir\cloud-wms-be-main_${company}_${yymmdd}.zip"

$headers = @{ Authorization = "token $token" }

Write-Host "BE 다운로드 중: $beZipUrl"
Invoke-WebRequest -Uri $beZipUrl -Headers $headers -OutFile $beOutPath -UseBasicParsing
Write-Host "BE 저장 완료: $beOutPath"
```

### 3단계 — FE 소스 다운로드

```powershell
$feRepo    = ($feUrl -replace '\.git$','') -replace 'https://github.com/',''
$feZipUrl  = "https://codeload.github.com/$feRepo/zip/refs/heads/$branch"
$feOutPath = "$outputDir\cloud-wms-fe-main_${company}_${yymmdd}.zip"

Write-Host "FE 다운로드 중: $feZipUrl"
Invoke-WebRequest -Uri $feZipUrl -Headers $headers -OutFile $feOutPath -UseBasicParsing
Write-Host "FE 저장 완료: $feOutPath"
```

### 4단계 — 파일 크기 확인

```powershell
$beSize = [math]::Round((Get-Item $beOutPath).Length / 1KB, 1)
$feSize = [math]::Round((Get-Item $feOutPath).Length / 1KB, 1)
```
