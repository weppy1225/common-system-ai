# 도메인 (Domain)

## 1. 시스템키

> **seq 타입 규칙**: 헤더(H) 테이블 seq = `int4` / 자식(D1~Dn) 테이블 seq = `bigint`

| 도메인 | 데이터 타입 | 기본 값 | 주석 |
|--------|--------------|----------|------|
| 시스템키(헤더) | int4 | | 헤더(H) 테이블 PK — 부모 테이블 seq |
| 시스템키(자식) | bigint | | 자식(D1~Dn) 테이블 PK — 다건 데이터 seq |
| 시스템키 | int4 | | 시스템키 (단일 테이블, 헤더 기준) |
| 사업장 | int4 | | 사업장ID |
| 센터 | int4 | | 센터ID |
| 거래처 | int4 | | 거래처ID |
| 품목 | int4 | | 품목ID |
| 라벨용지 | int4 | | 라벨용지ID |
| 창고 | int4 | | 창고ID |
| 위치 | bigint | | 위치ID |
| 사용자 | int4 | | 사용자ID |
| 사업장설정 | int4 | | 매출처, 시퀀스 |
| 파일시퀀스 | int4 | | 파일시퀀스 |
| 요청SEQ | int4 | | 요청SEQ |
| 품목SEQ | bigint | | 품목SEQ |
| 처리SEQ | bigint | | 처리SEQ |
