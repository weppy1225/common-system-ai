## 고정 작성 규칙

### 검색 영역 배치 규칙
- 1행 5컬럼 기준 배치
- 사업장(ZSelect) 항상 **1행 1번** 고정
- 5개 초과 시 2행으로 내림
- 남는 칸: `(빈칸)` 으로 명시

### 컴포넌트 표기 기준

| 컴포넌트 종류 | 표기 |
|---|---|
| ZSelect (사업장·센터) | Select |
| ZCodeSelect (공통코드) | Select |
| ZText type="text" | Input |
| ZText type="number" | Input - 숫자 |
| ZCalendar | Date |
| ZProdPopup / 품목팝업 연동 | Input + 팝업 🔍 |
| ZTextArea | Textarea |

### 그리드 컬럼 정렬 기준

| 데이터 성격 | 정렬 표기 |
|---|---|
| 이름·명칭·설명 (Nm·NmRes) | 좌측 |
| 코드·번호·상태 (Cd·No) | 중앙 |
| 날짜 (Ymd) | 중앙 |
| 수량·금액·수치 (Qty·Amt·Val) | 우측 |
| 체크여부 (useYn, 아이콘 표시) | 중앙 |

### 그리드 컬럼 작성 규칙
- No. / checkbox 항상 첫 2행 (순번 1, 2)
- PK·숨김 컬럼 (Seq류, visible:false) 목록 **제외**
- 비고란: 숫자·날짜 포맷 / 조건부 편집 / 팝업 연동 / 자동채움 등 명시

### UI유형 표기 기준

| 구조 | 표기 |
|---|---|
| 그리드 1개 | 단일 그리드 |
| 상단 헤더 + 하단 디테일 | 그리드 2개 상단 하단 |
| 상단 + 하단 좌우 분할 | 그리드 3개 상단 하단좌측 하단우측 |

### 업무규칙 구분값 (고정)

| 구분 | 설명 |
|---|---|
| 순서강제 | A 등록 후 B 등록 가능 |
| 수량제한 | N개 초과 등록 불가 |
| 값강제 | X값이면 Y값 고정 |
| UI disable | 조건에 따라 입력 비활성화 |
| 특수기능 | 금액복사·자동계산 등 |

### 팝업 여부 판단 기준
- 등록/수정 필드 **6개 이상** → Edt.vue 팝업 (N절 작성)
- 필드 5개 이하 단순 처리 → 팝업 절 생략
