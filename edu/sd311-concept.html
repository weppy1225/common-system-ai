스킬명

SD_311 (화면설계 프로토타입 HTML 생성)

실행: /SD_311 {메뉴코드}

2

주요 작업 내용

화면요건 문서(ui.md)를 읽어 프로토타입 HTML + 목업 데이터 JS를 생성하고 좌측 메뉴에 자동 등록한다.

3

스킬 개요

요건 문서(ui.md)를 공통 CSS/JS·정해진 템플릿에 채워 실행 가능한 프로토타입 화면으로 변환하는 스킬이다. 공통 자산은 절대 수정하지 않고 <link>/<script> 로 참조하며, 템플릿의 플레이스홀더만 치환한다.

입력 ui.md 가 단일 진실원천(SoT) 이므로, 화면을 바꿀 땐 화면을 직접 고치지 않고 ui.md 를 수정한 뒤 이 스킬을 다시 실행한다.

4

육하원칙

누가화면설계 담당자가 Claude와 함께 (AI가 템플릿 변환을 수행)
언제ui.md(SD_310_UI 산출물)가 준비된 뒤
어디서AI 허브 레포(common-system-ai) 폴더 안에서 /SD_311 {메뉴코드} 실행
무엇을ui.md 의 화면요건(UI유형·검색·그리드·팝업·업무규칙)을
어떻게공통 자산 참조 + 템플릿 플레이스홀더 치환으로 (공통 CSS/JS 수정 없이)
왜요건을 눈으로 검증할 실행 화면을 만들고 좌측 메뉴에 연결하기 위해
