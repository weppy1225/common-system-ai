# 구분 단계명 처리 내용 비고
1 트리거 슬래시 명령 실행 사용자가 /PS-doc-copy 입력 Claude Code 슬래시 커맨드
2 정보 수집 프로젝트 코드 입력 사용자에게 프로젝트 코드를 요청 후 수집
예: PJWMS001
3 정보 수집 고객사명 입력 사용자에게 고객사명을 요청 후 수집
예: (주)테스트물류
4 처리 출력 경로 결정 수집된 정보를 조합하여 대상 경로 생성
Z:\DEV_PROJECT\{코드} {고객사명}-WMS 구축		 경로 중복 시 사용자 확인
5 처리 GitHub Token 로드 .claude/config/github.json에서 PAT 읽기 Private repo 접근 인증
6 처리 Template ZIP 다운로드 Invoke-WebRequest로 GitHub에서
project-doc-template.git ZIP 다운로드		 입력: https://github.com/ZinCoAtoZ/project-doc-template.git
7 처리 ZIP 압축 해제 다운로드된 ZIP을 임시 경로에 압축 해제
8 처리 폴더 복사 압축 해제된 폴더·파일을 대상 경로로 복사 출력: Z:\DEV_PROJECT\{코드} {고객사명}-WMS 구축\
9 완료 보고 결과 보고 생성된 폴더 트리 구조를 출력하고 완료 메시지 표시
입출력 요약
구분내용
입력 프로젝트 코드 (사용자 입력)
고객사명 (사용자 입력)
GitHub Template URL: https://github.com/ZinCoAtoZ/project-doc-template.git
GitHub PAT: .claude/config/github.json
출력 Z:\DEV_PROJECT\{프로젝트코드} {고객사명}-WMS 구축\ — 문서 폴더 전체
