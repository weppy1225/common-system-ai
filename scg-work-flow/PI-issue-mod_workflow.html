#구분단계명처리 내용비고
1 트리거 git commit 실행 개발자가 git commit 실행 시 자동 발동
git hook(post-commit) 또는 커밋 메시지 입력 후 Claude 호출 		커밋 타이밍에 자동 실행
2 처리 커밋 메시지 파싱 커밋 메시지에서 Redmine 이슈 번호 추출
예: #1234, refs #1234, fixes #1234
진행 상태 키워드 추출 (진행중 / 완료 / 리뷰요청) 		이슈 번호 없으면 수정 없이 종료
3 처리 현재 이슈 상태 조회 GET /issues/{id}.json 호출
현재 상태 / 담당자 / 진행률 확인 		Redmine API Key: .claude/config/redmine.json
4 처리 이슈 상태 수정 키워드에 따라 이슈 필드 자동 업데이트:
· 진행중 → 상태: 진행중, 진행률: 50%
· 완료 → 상태: 완료, 진행률: 100%
· 리뷰요청 → 상태: 검토중, 검토자 지정 		PUT /issues/{id}.json
5 완료 보고 결과 보고 수정된 이슈 번호 + 변경 전/후 상태 출력
Redmine 이슈 URL 링크 제공
커밋 메시지 패턴 → 이슈 수정 결과
# 예시 커밋 메시지

fix: 출고지시 저장 버그 수정 refs #1235 완료
refactor: 재고 모듈 리팩토링 #1236 리뷰요청
