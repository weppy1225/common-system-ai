# common-system-ai

업무 시스템 AI 프레임워크 허브 레포. 화면설계·지식베이스·소스패턴·산출물·BE/FE 개발 자동화 스킬의 단일 허브.

## 무엇이 어디에

`knowledgebase/`(공통 배경) · `spec/{프로젝트}/{메뉴}/`(메뉴 설계 정본) · `prototype/`(검증 화면) · `patterns/`(소스 패턴) · `deliverables/`(산출물) · `.claude/{skills,rules}/`(스킬·규칙)

> 폴더별 역할·읽는 사람·경계 규칙(정본)은 → [`STRUCTURE.md`](./STRUCTURE.md). 여기엔 중복하지 않는다.
