# MDBZ01 화면 구조 (UI 명세)

## 1. 화면 목록

| 화면명 | 화면코드 | 형태 | 설명 |
|---|---|---|---|
| 사업장 | MDBZ01 | 메인 화면 (좌우 2분할) | 사업장 기본정보(좌) + 물류센터 그리드(우) |
| 대행의뢰센터 정보수정 | MDBZ01P01 | 팝업 | 자사 센터의 물류대행 여부 및 주소 설정 |
| 물류대행업체 검색 | MDBZ01P02 | 팝업 | 외부 물류대행업체 검색 및 위탁 의뢰 신청 |

## 2. 메인 화면

### 2-1. 전체 레이아웃

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 헤더: 사업장 [MDBZ01]                             [물류센터] [저장]  [PC매뉴얼] │
├────────────────────────────┬────────────────────────────────────────────┤
│                            │                                            │
│  [사업장 기본정보 영역]      │         [물류센터 그리드 영역]              │
│                            │                                            │
│  사업장     [드롭다운]       │  물류센터명 / 우편번호 / 주소 / 주소상세     │
│  사업장명*  [텍스트 입력]    │  전화번호 / 비고 / 사용여부                 │
│  대표자     [텍스트 입력]    │                                            │
│  사업자등록번호 [텍스트]     │  (행 추가 / 편집 / 삭제 가능)              │
│  업태       [텍스트 입력]    │                                            │
│  업종       [텍스트 입력]    │                                            │
│  우편번호   [텍스트+검색]    │                                            │
│  주소       [읽기전용]       │                                            │
│  주소상세   [텍스트 입력]    │                                            │
│  전화번호   [텍스트 입력]    │                                            │
│  이메일     [텍스트 입력]    │                                            │
│  사업장구분  [표시 전용]     │                                            │
│  사용여부   [표시 전용]      │                                            │
│  사업장이미지 [파일 업로드]  │                                            │
│  사업장테마색 [색상 선택]    │                                            │
│                            │                                            │
│  [저장]                     │                                            │
└────────────────────────────┴────────────────────────────────────────────┘
```

- 화면은 좌측(사업장 기본정보, 약 35% 너비)과 우측(물류센터 그리드, 약 65% 너비)으로 2분할된다.
- 헤더 영역: 좌측에 "사업장 [MDBZ01]" 타이틀, 우측에 "물류센터" 섹션 타이틀 + 저장 버튼 + PC 매뉴얼 버튼

### 2-2. 사업장 기본정보 항목

| 순번 | 항목명 | 입력유형 | 필수 | 비고 |
|---|---|---|---|---|
| 1 | 사업장 | 드롭다운 선택 | - | 수정 가능한 사업장 목록. 선택 변경 시 우측 정보 갱신 |
| 2 | 사업장명 | 텍스트 입력 | ✓ | 필수. 비어 있으면 저장 거부 |
| 3 | 대표자 | 텍스트 입력 | - | |
| 4 | 사업자등록번호 | 텍스트 입력 | - | |
| 5 | 업태 | 텍스트 입력 | - | |
| 6 | 업종 | 텍스트 입력 | - | |
| 7 | 우편번호 | 텍스트 + 주소 검색 | - | 클릭 시 주소 검색 팝업 호출. 읽기전용 필드 |
| 8 | 주소 | 표시 전용 | - | 주소 검색 결과 자동 입력. 직접 입력 불가 |
| 9 | 주소상세 | 텍스트 입력 | - | |
| 10 | 전화번호 | 텍스트 입력 | - | |
| 11 | 이메일 | 텍스트 입력 | - | |
| 12 | 사업장구분 | 표시 전용 | - | 자사물류/물류대행 표시. 변경 불가(고객문의 필요) |
| 13 | 사용여부 | 표시 전용 | - | 사용/미사용 텍스트 표시 |
| 14 | 사업장이미지 | 파일 업로드 | - | 이미지 추가·삭제 가능. 권장 사이즈 149×40 |
| 15 | 사업장 테마색 | 색상 선택 | - | 색상 선택기(컬러피커)로 입력 |

### 2-3. 물류센터 그리드 컬럼

| 컬럼명 | 정렬 | 편집 가능 | 비고 |
|---|---|---|---|
| 물류센터명 | 좌측 | 가능 (신규 행만) | 위탁 센터는 편집 불가 |
| 우편번호 | 중앙 | 불가 (검색 아이콘으로 입력) | 위탁 센터는 검색 아이콘 미표시 |
| 주소 | 좌측 | 불가 | 우편번호 검색 결과 자동 입력 |
| 주소상세 | 좌측 | 가능 | |
| 전화번호 | 중앙 | 가능 | |
| 비고 | 좌측 | 가능 | |
| 사용여부 | 중앙 | 불가 | 사용/미사용 텍스트 표시 |
| centerSeq | — | — | (내부 처리용, 목록 미표시) |

### 3-2. MDBZ01P02 — 물류대행업체 검색

#### 검색 조건

| 순번 | 항목명 | 입력유형 | 필수 | 비고 |
|---|---|---|---|---|
| 1 | 사업장 | 텍스트 입력 | - | 사업장명 부분 검색 |
| 2 | 물류센터 | 텍스트 입력 | - | 센터명 부분 검색 |
| 3 | 주소 | 텍스트 입력 | - | 주소 부분 검색 |

#### 검색 결과 그리드 컬럼

| 컬럼명 | 정렬 | 비고 |
|---|---|---|
| (체크박스) | 중앙 | 다중 선택 가능 |
| 요청상태 | 중앙 | 신청 가능 / 신청중 / 승인 / 거절 표시 |
| 사업장명 | 좌측 | 동일 사업장 셀 병합 표시 |
| 센터명 | 좌측 | |
| 주소 | 좌측 | 주소 + 주소상세 합쳐서 표시 |
| 전화번호 | 중앙 | |
| 이메일 | 중앙 | |
| bizSeq | — | (내부 처리용, 목록 미표시) |
| centerSeq | — | (내부 처리용, 목록 미표시) |

#### 의뢰 신청 정보 입력 영역

| 항목명 | 입력유형 | 비고 |
|---|---|---|
| 사업장 | 표시 전용 | 현재 로그인 사용자의 사업장명 |
| 담당자 | 표시 전용 | 현재 로그인 사용자명 |
| 전화번호 | 표시 전용 | 현재 로그인 사용자 전화번호 |
| 요청내용 | 텍스트 영역 | 위탁 의뢰 메시지 입력 |

#### 버튼 목록

| 버튼명 | 조건 | 동작 설명 |
|---|---|---|
| 검색 | 항상 표시 | 입력한 조건으로 물류대행업체 검색 |
| 초기화 | 항상 표시 | 검색 조건 초기화 |
| 위탁 요청 | 항상 표시 | 선택된 센터에 위탁 의뢰 신청. 선택 없으면 경고 메시지 표시 |

## 4. 화면 간 연동 요약

| 사용자 액션 | 결과 |
|---|---|
| 사업장 드롭다운에서 다른 사업장 선택 | 사업장 기본정보 및 물류센터 그리드가 선택된 사업장 데이터로 갱신됨 |
| 사업장 기본정보 저장 성공 | 화면이 갱신되고 시스템 헤더의 사업장명이 즉시 업데이트됨 |
| 물류센터 저장 성공 | 센터 그리드가 최신 목록으로 갱신됨 |
| MDBZ01P01 저장 완료 | 팝업이 닫히고 메인 화면의 센터 그리드가 자동 갱신됨 |
| MDBZ01P02 위탁 요청 성공 | 팝업이 닫히고 메인 화면의 센터 그리드가 자동 갱신됨 |
