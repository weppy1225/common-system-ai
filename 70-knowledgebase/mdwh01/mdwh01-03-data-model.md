# MDWH01 데이터 모델 (테이블·상태값)

## 1. 물리 테이블 목록

| 업무 개념 | 물리 테이블명 | 업무 역할 |
|---|---|---|
| 창고 | MDM_WH | 창고의 기본 정보(명칭·그룹·처리기능·온도·사용여부 등)를 저장하는 핵심 마스터 테이블 |
| 물류센터 | MDM_CENTER | 창고가 속한 물류센터 정보를 제공하는 참조 테이블 |
| 사업장-센터 매핑 | MDM_BIZ_CENTER | 사업장과 물류센터 간의 계약/위탁 관계를 저장하는 매핑 테이블 |
| 사업장-창고 매핑 | MDM_BIZ_WH | 사업장과 창고 간의 사용 관계 및 외부 연동 식별자(IF창고ID)를 저장하는 매핑 테이블 |
| 위치(로케이션) | MDM_LOC | 창고 내 물리적 위치(로케이션) 정보를 저장하는 테이블. 창고 등록 시 기본 위치가 자동 생성됨 |
| 공통코드 상세 | SM_COMM_D | 창고그룹명·사용여부명·온도구분명 등 코드 변환에 사용하는 공통코드 테이블 |
| 재고 수불 | WMS_INVEN_INOUT | 재고의 입출고 이력을 저장하는 거래 테이블. 창고 삭제 가능 여부 판단에 사용됨 |

## 2. 상태값 / 코드 규칙

### 2-1. 사용여부 (use_yn)

| 코드값 | 의미 | 비고 |
|---|---|---|
| Y | 사용 | 창고가 정상 운영 중 |
| N | 미사용 | 창고 운영 중단 상태 |

```
[등록]
  │
  ▼
 [Y: 사용] ──────────────→ [N: 미사용]
  ▲                              │
  └──────────────────────────────┘
         (다시 사용으로 변경 가능)
```

### 2-2. 처리기능 플래그 (각 컬럼별 Y/N)

각 처리기능은 독립적인 Y/N 플래그 컬럼으로 관리된다.

| 컬럼 | 기능명 | Y 의미 |
|---|---|---|
| in_yn | 입고(to) | 이 창고로 상품을 입고 처리할 수 있음 |
| pick_yn | 출고(fr) | 이 창고에서 피킹(출고 집품) 처리할 수 있음 |
| out_yn | 출하(fr) | 이 창고에서 출하 처리할 수 있음 |
| return_yn | 반품(to) | 이 창고로 반품 입고 처리할 수 있음 |
| etc_yn | 예외(fr) | 이 창고에서 예외 출고 처리할 수 있음 |
| st_yn | 세트작업 | 이 창고에서 세트 구성/해체 작업을 수행할 수 있음 |
| rp_yn | 전환처리 | 이 창고에서 재고 전환 처리를 수행할 수 있음 |

### 2-3. 기본창고 여부 (def_wh_yn)

| 코드값 | 의미 |
|---|---|
| Y | 해당 센터의 기본 창고로 지정됨 (삭제 불가) |
| N | 일반 창고 |

### 2-4. 가용재고 여부 (available_inven_yn)

| 코드값 | 의미 |
|---|---|
| Y | 이 창고의 재고를 주문 가용 재고로 집계함 |
| N | 주문 가용 재고 집계에서 제외 |

### 2-5. 기본 위치 여부 (def_loc_yn, MDM_LOC 테이블)

| 코드값 | 의미 |
|---|---|
| Y | 창고 등록 시 자동 생성된 기본 위치 (창고명 + "BL") |
| N | 수동으로 추가된 일반 위치 |

### 2-6. 창고그룹 코드 (wh_group_cd)

창고그룹 코드는 공통코드 WH_GROUP_CD로 관리된다. 소스에서 확인된 코드값은 아래와 같다.

| 코드값 | 창고그룹명 | 기본 처리기능 |
|---|---|---|
| G10 | 입고그룹 | 입고(to) |
| G20 | 반품그룹 | 반품(to) |
| G30 | 제품그룹 | 출고(fr) |
| G50 | 출고그룹 | 출하(fr) |
| G70 | 외주그룹 | 입고·반품·출하·예외·세트·전환 |
| G90 | 예외그룹 | 예외·세트 |

### 2-7. 위탁(물류대행) 판별 조건 (MDM_BIZ_CENTER)

사업장-센터 매핑 테이블에서 등록 사업장과 현재 사업장이 다른 경우 위탁 관계로 판별된다.

| 조건 | 의미 |
|---|---|
| biz_seq = reg_biz_seq | 센터를 직접 소유한 사업장 |
| biz_seq ≠ reg_biz_seq (AND 승인 완료) | 센터를 위탁 이용 중인 사업장 |
