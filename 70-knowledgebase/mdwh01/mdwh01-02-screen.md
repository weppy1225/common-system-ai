# MDWH01 화면 구조 (UI 명세)

## 1. 화면 목록

| 화면명 | 화면코드 | 형태 | 설명 |
|---|---|---|---|
| 창고 목록 | MDWH01 | 메인 화면 | 창고 검색 조건 입력 및 목록 조회 |
| 창고 등록 | MDWH01P01 | 팝업 | 신규 창고 등록 폼 |
| 창고 수정 | MDWH01P02 | 팝업 | 기존 창고 정보 수정 폼 (등록 팝업 재사용) |

## 2. 메인 화면

### 2-1. 전체 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  창고 [MDWH01] ∨  [검색] [초기화]         [PC매뉴얼] │
├─────────────────────────────────────────────────────┤
│  [물류센터: 드롭다운] [창고그룹: 다중선택]             │
│  [창고명: 텍스트입력] [사용여부: 드롭다운]  [     ]    │
├─────────────────────────────────────────────────────┤
│  [추가] [수정] [삭제]                                │
├─────────────────────────────────────────────────────┤
│  창고 목록 그리드                                    │
│  (물류센터 / 창고그룹 / 창고명 / IF창고ID /           │
│   가용여부 / 온도구분 / 처리기능 그룹 / 사용여부)      │
├─────────────────────────────────────────────────────┤
│  [페이징]                                           │
└─────────────────────────────────────────────────────┘
```

### 2-2. 검색 조건

| 순번 | 항목명 | 입력유형 | 필수 | 비고 |
|---|---|---|---|---|
| 1 | 물류센터 | 드롭다운 선택 | - | 사업장에 연결된 센터 목록, 기본값: 전체 |
| 2 | 창고그룹 | 다중 선택 | - | 창고그룹 코드 목록에서 복수 선택 가능 |
| 3 | 창고명 | 텍스트 입력 | - | 부분 일치 검색 |
| 4 | 사용여부 | 드롭다운 선택 | - | 전체/사용/미사용, 기본값: 사용 |

- 검색 초기화 시 사용여부는 "사용"으로 초기화된다.
- 엔터키로도 검색이 실행된다.

### 2-3. 목록 컬럼

| 컬럼명 | 정렬 | 비고 |
|---|---|---|
| 물류센터 | 좌측 | 창고가 속한 센터명 |
| 창고그룹 | 중앙 | 창고그룹명 (코드 변환 표시) |
| 창고명 | 좌측 | 창고 명칭 |
| IF창고ID | 좌측 | 외부 연동용 창고 식별자 |
| 가용여부 | 중앙 | 가용재고 여부 (코드 변환 표시) |
| 온도구분 | 중앙 | 창고 온도 환경 (코드 변환 표시) |
| 처리기능 > 입고(to) | 중앙 | 기능 보유 시 체크 아이콘 표시 |
| 처리기능 > 출고(fr) | 중앙 | 기능 보유 시 체크 아이콘 표시 |
| 처리기능 > 출하(fr) | 중앙 | 기능 보유 시 체크 아이콘 표시 |
| 처리기능 > 반품(to) | 중앙 | 기능 보유 시 체크 아이콘 표시 |
| 처리기능 > 예외(fr) | 중앙 | 기능 보유 시 체크 아이콘 표시 |
| 처리기능 > 세트 | 중앙 | 기능 보유 시 체크 아이콘 표시 |
| 사용여부 | 중앙 | 사용/미사용 (코드 변환 표시) |
| 기본창고여부 | - | (내부 처리용, 목록 미표시) — 삭제 차단 조건에 활용 |
| 물류대행센터여부 | - | (내부 처리용, 목록 미표시) — 수정 버튼 활성화 조건에 활용 |

- "처리기능" 헤더 아래 6개 하위 컬럼이 그룹으로 묶여 표시된다.
- 컬럼 필터 기능 지원 (각 컬럼 헤더에서 필터 적용 가능).
- 행 체크박스 선택 지원 (다중 선택 가능).
- 행 더블클릭 시 수정 팝업이 열린다 (위탁 사업장 담당자는 더블클릭 무시).

### 2-4. 버튼 목록

| 버튼명 | 표시 조건 | 동작 |
|---|---|---|
| 추가 | 등록 사업장 담당자에게만 표시 | 창고 등록 팝업(MDWH01P01) 오픈 |
| 수정 | 등록 사업장 담당자에게만 표시 | 체크된 창고 1건 선택 후 수정 팝업(MDWH01P02) 오픈 |
| 삭제 | 등록 사업장 담당자에게만 표시 | 체크된 창고 1건 이상 선택 후 삭제 확인 후 실행 |
| PC매뉴얼 | 항상 표시 | PC 사용자 매뉴얼 오픈 |

- 조회 결과가 위탁 사업장 소속 창고인 경우 추가·수정·삭제 버튼이 모두 숨겨진다.
- 수정 버튼은 체크된 항목이 0건이면 미선택 안내, 2건 이상이면 1건만 선택 안내를 표시한다.
- 삭제 버튼은 체크된 항목 중 기본창고가 포함된 경우 삭제 불가 안내를 먼저 표시한다.

## 3. 창고 등록/수정 팝업 (MDWH01P01 / MDWH01P02)

### 3-1. 폼 항목

| 순번 | 항목명 | 입력유형 | 필수 | 등록 가능 | 수정 가능 | 비고 |
|---|---|---|---|---|---|---|
| 1 | 물류센터 | 드롭다운 선택 | 필수 | 가능 | 불가 | 등록 후 변경 불가 (비활성) |
| 2 | 창고그룹 | 드롭다운 선택 | 필수 | 가능 | 불가 | 등록 후 변경 불가 (비활성), 선택 시 처리기능 자동 설정 |
| 3 | 창고명 | 텍스트 입력 | 필수 | 가능 | 가능 | 최대 30자, 위탁 사업장 창고는 수정 불가 |
| 4 | IF창고ID | 드롭다운 선택 | - | 가능 | 가능 | 외부 ERP 연동 창고코드 선택 |
| 5 | 가용재고 | 라디오 버튼 선택 | - | 가능 | 조건부 | 가용/비가용 선택, 창고그룹에 따라 변경 불가 |
| 6 | 온도구분 | 드롭다운 선택 | 필수 | 가능 | 조건부 | 상온 기본값(D), 위탁 사업장 창고는 수정 불가 |
| 7 | 처리기능 | 체크박스 다중 선택 | 필수(1개 이상) | 가능 | 조건부 | 입고/반품/출고(fr)/출하(fr)/예외/세트 선택, 창고그룹에 따라 일부 고정 |
| 8 | 사용여부 | 드롭다운 선택 | - | 미표시 | 가능 | 수정 모드에서만 표시, 창고그룹에 따라 변경 불가 |

- 처리기능 중 창고그룹에 따라 변경이 불가한 항목은 비활성(취소선 레이블) 표시된다.
- 위탁 사업장 소속 창고의 수정 팝업은 모든 항목이 비활성 처리된다.

### 3-2. 버튼 목록

| 버튼명 | 조건 | 동작 |
|---|---|---|
| 등록 | 등록 모드(MDWH01P01) | 유효성 검증 후 창고 신규 저장 |
| 수정 | 수정 모드(MDWH01P02) | 유효성 검증 후 창고 정보 갱신 |

- 저장 성공 시 팝업이 닫히고 목록이 자동 갱신된다.

## 4. 화면 간 연동 요약

| 사용자 액션 | 결과 |
|---|---|
| 메인 화면 검색 | 사업장에 속한 창고 목록 갱신 |
| 추가 버튼 클릭 | 창고 등록 팝업 오픈 (물류센터 검색 조건 선택값 자동 설정) |
| 수정 버튼 클릭 / 행 더블클릭 | 창고 단건 조회 후 수정 팝업 오픈 |
| 등록 팝업 저장 | 팝업 닫힘 + 메인 목록 자동 갱신 |
| 수정 팝업 저장 | 팝업 닫힘 + 메인 목록 자동 갱신 |
| 삭제 버튼 클릭 | 삭제 확인 → 삭제 실행 → 메인 목록 자동 갱신 |
| 창고그룹 변경 | 처리기능 항목 자동 재설정 및 수정 가능 항목 재계산 |
